4. Write Back

Simpan hasil ke register, memori, atau perangkat I/O agar tersedia untuk instruksi berikutnya.

Komponen utama
  • PC (Program Counter): alamat instruksi berikutnya.
  • IR (Instruction Register): menyimpan instruksi sekarang.
  • CU (Control Unit): mengatur & decode instruksi.
  • ALU: melakukan operasi aritmetika & logika.
Ringkasan singkat

Siklus berulang terus selama program berjalan: CPU fetch instruksi, decode, execute, write back. Banyak arsitektur modern menambahkan tahapan pipelining dan optimisasi agar beberapa instruksi dapat diproses secara bersamaan.

Fetch -> Decode -> Execute -> Write Back
(PC update) -> Fetch next instruction
